# my_world - Gazebo室内仿真地图包

## 1. 地图用途
用于ROS Noetic + Gazebo环境下的室内服务机器人导航与对比实验。
重点测试差速轮与麦克纳姆轮（全向轮）在不同场景下的运动性能差异。

## 2. 地图尺寸
- 总长度：12m
- 总宽度：10m
- 地面高度：z = 0

## 3. 坐标系说明
- world原点：地图左下角 (0, 0)
- x轴正方向：从左向右（东）
- y轴正方向：从下向上（北）
- 地面高度：z = 0

## 4. 推荐机器人出生点
- 位置：x = 1.5, y = 9.0, yaw = 0.0
- 起点/充电区

## 5. 推荐导航目标点
| 目标点 | 坐标 (x, y) | 说明 |
| A | (5.0, 9.0) | 上方服务区 |
| B | (8.0, 9.0) | 上方服务区 |
| C | (6.0, 5.5) | L型走廊附近 |
| D | (10.0, 5.5) | U型服务区内 |
| 货架区 | (1.5, 2.5) | 侧向对接 |
| 停靠区 | (6.0, 2.5) | 平行停车 |

## 6. 测试场景
| 场景 | 位置 | 尺寸 | 测试重点 |
| 窄走廊避障 | (3, 6.5) | 宽0.8m | 差速轮需倒车，全向轮直接横移 |
| L型走廊 | (6, 6.5) | 宽0.65m | 差速轮原地旋转，全向轮斜向平移 |
| U型服务区 | (10, 6.5) | 上宽0.65m，侧宽1.15m | 差速轮多次转向，全向轮横移回转 |
| 货架区 | (1.5, 2.5) | - | 侧向对接，全向轮直接横移停靠 |
| 停靠区 | (6.0, 2.5) | 障碍物间距2.6m | 平行停车，全向轮直接侧移入位 |

## 7. 是否提供导航地图
- [ ] 已提供pgm/yaml导航地图
- [x] 未提供，需整合负责人后续通过gmapping/cartographer建图
